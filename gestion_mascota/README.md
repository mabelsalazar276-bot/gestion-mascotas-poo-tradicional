# Sistema de Gestión de Mascotas

Este repositorio contiene la solución a la actividad práctica de gestión de mascotas, desarrollada bajo dos enfoques o paradigmas de programación distintos utilizando Python.

## Estructura del Proyecto

El código se encuentra organizado de la siguiente manera:

* **programacion_tradicional/**: Contiene el archivo `tradicional.py`. Resuelve el problema mediante el uso exclusivo de funciones, variables y captura de datos por teclado.
* **programacion_poo/**: Contiene los archivos `mascota.py` y `main.py`. Resuelve el problema mediante Programación Orientada a Objetos, evidenciando el uso de clases, atributos, métodos y abstracción.

## Cómo Ejecutar los Programas

### 1. Ejecutar el Enfoque Tradicional
Abre la terminal y ejecuta:
```bash
python programacion_tradicional/tradicional.py
```

### 2. Ejecutar el Enfoque Orientado a Objetos
Abre la terminal y ejecuta:
```bash
python programacion_poo/main.py
```
